# Build an engagement sequence

> Set up a repeatable outreach flow once, then enroll candidates into it as you find them.

## Before you start
- Decide on the channels you'll use — LinkedIn, email, or a mix.
- Have a draft of your first message; you can edit later.

## Steps

1. Go to **AI Sourcing → Sequences** and click **New sequence**. Give it a name and short description so teammates know when to use it.
   ![New sequence form](./assets/01-new-sequence.png)
2. Add steps in order. For each step choose the channel, the day offset (e.g. *Day 0*, *Day 3*, *Day 7*), and the message template. Add a subject if the channel needs one.
   ![Sequence steps editor](./assets/02-steps.png)
3. Optionally add an **AI prompt hint** to each step. Tech1m uses it later to personalize the message for each candidate.
4. Toggle the sequence to **Active** when you're ready. Then enroll candidates from a search, a saved search, or a candidate profile.
   ![Enroll candidates dialog](./assets/03-enroll.png)

## Tips
- Keep sequences short — three to five steps is usually enough.
- Use offset days to space outreach: *Day 0* connect, *Day 3* follow up, *Day 7* break-up message.
- Duplicate a successful sequence and tweak it for different roles instead of starting over.

## Troubleshooting
- **Can't enroll candidates** — the sequence is paused. Toggle it to Active and try again.
- **Can't delete a sequence** — it has active enrollments. Pause it or wait for enrollments to finish first.
