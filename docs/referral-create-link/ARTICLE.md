# Create a referral link

> Get a unique link you can share with peers — when they sign up and qualify, you earn rewards.

## Before you start
- You need access to the referral program. It's available to all active workspaces by default.
- Read the program terms once so you know which actions trigger a reward.

## Steps

1. Open **Refer & earn** from the sidebar (or your profile menu). The dashboard shows your current rewards and a default referral code.
   ![Referral dashboard](./assets/01-referral-dashboard.png)
2. Click **Create link**. You can leave the default code or customise it — for example, a campaign name like `summer-launch`. Then click **Save**.
   ![Create referral link](./assets/02-create-link.png)
3. Use **Copy link** or one of the share buttons to send it via email, LinkedIn, or your channel of choice. Each click is attributed to your account.
   ![Share referral link](./assets/03-share-link.png)

## Tips
- Create a separate link per channel or campaign so you can see what works best.
- Add a short personal note when you share — it consistently lifts conversion.

## Troubleshooting
- **The custom code is taken** — codes are unique across Tech1m. Try a variation.
- **My link doesn't work** — make sure you copied the full URL. Test it in a private browser window before sharing widely.
