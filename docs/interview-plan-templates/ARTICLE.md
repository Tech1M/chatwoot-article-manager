# Build interview plan templates

> An interview plan tells evaluators what to ask, in what order, and how to score - all reusable across hires.

## Before you start
- You need admin or hiring manager access on the role.
- A scorecard template that matches the competencies you want to evaluate.

## Steps

1. Go to **Interviews -> Plans** and click **New plan**. Give it a name and link the job or job family.
   ![New plan](./assets/01-new-plan.png)
2. Add sections in order: for example, intro, technical, behavioral, candidate Q&A. Set a duration for each.
3. For each section, pull questions from the question bank or write inline questions. Add notes for the interviewer.
   ![Plan sections](./assets/02-sections.png)
4. Attach a scorecard template so evaluators rate the right competencies.
5. Click **Publish**. The plan is now selectable when scheduling an interview for that role.

## Tips
- Duplicate an existing plan to spin up role variations quickly.
- Use **Branching** to swap sections based on candidate seniority.

## Troubleshooting
- **Plan does not appear when scheduling** — the plan is saved as a draft. Open it and click **Publish**.
- **Scorecard does not match the plan** — open the plan, swap the scorecard template, and republish. Already-scheduled interviews keep their original scorecard.
