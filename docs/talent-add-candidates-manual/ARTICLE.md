# Add a candidate manually

> Use this when you have a single candidate to add — for example, someone you met at an event or were referred to by a teammate.

## Before you start
- Create the talent pool you want to add the candidate to.
- Have the candidate's name and contact details (email or phone) ready.

## Steps

1. Open the talent pool you want to add the candidate to.
   ![Pool overview](./assets/01-pool-overview.png)
2. Click **Add candidate** and choose **Add manually** from the menu.
   ![Add manually option](./assets/02-add-manually.png)
3. Fill in the candidate's details: name, email, phone, location, current role, and a short summary. Email or phone is required so you can reach out later.
   ![Candidate form](./assets/03-candidate-form.png)
4. Optionally attach a resume, add tags, or mark the candidate as a favorite.
5. Click **Save**. The candidate appears in the pool right away.
   ![Candidate added](./assets/04-candidate-added.png)

## Tips
- If the system warns you that a similar profile already exists, open it before creating a duplicate. You can still add the existing candidate to this pool.
- The more detail you add now, the better search and AI suggestions will work later.

## Troubleshooting
- **Save is disabled** — check that name and at least one contact field are filled in.
- **"Candidate already exists" message** — click through to the existing record and add it to the pool instead of creating a duplicate.
