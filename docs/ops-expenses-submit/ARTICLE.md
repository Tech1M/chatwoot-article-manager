# Submit an expense

> Log money you have spent on behalf of a project, attach the receipt, and send it for approval.

## Before you start
- You have a clear receipt or invoice (PDF or image).
- You know which project the expense should be charged to.
- The expense type you need exists; if not, ask an admin to add it.

## Steps

1. Open **Operations → Expenses** and click **New expense**.
   ![Expenses page with new expense button](./assets/01-expenses-page.png)
2. Fill in the date, amount, currency, and pick the expense type and project. Add a short description so the approver can place the spend in context.
   ![New expense form fields](./assets/02-expense-form.png)
3. Attach the receipt. The file is stored with the expense and stays available for accounting later.
   ![Receipt upload area on the expense form](./assets/03-receipt-upload.png)
4. Submit the expense. It moves to **Pending** and the project's approver is notified.

## Tips
- Submit expenses promptly — many companies have a cutoff per period.
- If the receipt is not in your home currency, the system records both the original amount and the converted value.

## Troubleshooting
- **My project is not in the dropdown** — ask the project owner to add you, then reopen the form.
- **Receipt fails to upload** — the file may be too large or in an unsupported format. Try a JPG or PDF under the size limit.
