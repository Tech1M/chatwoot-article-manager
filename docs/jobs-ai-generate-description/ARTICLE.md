# Generate a job description with AI

> Skip the blank page — let the AI draft your job description and edit from there.

## Before you start
- Have a clear idea of the role title, seniority, and key skills.
- Decide whether the role is remote, hybrid, or on-site.

## Steps

1. While creating or editing a job, open the description step and click **Generate with AI**.
   ![Generate with AI button](./assets/01-generate-button.png)
2. Fill in the prompt fields — title, seniority, key skills, location, and any extra context — then click **Generate**.
   ![AI prompt panel](./assets/02-prompt.png)
3. Review the draft. If it isn't quite right, click **Regenerate** or tweak the prompt and try again.
   ![Generated draft preview](./assets/03-draft.png)
4. Click **Use this draft** to drop the content into your job description, then edit any sections directly in the rich-text editor.

## Tips
- Add 3–5 specific skills in the prompt — vague prompts produce generic copy.
- Mention your company's voice ("warm and casual", "formal and direct") to shape the result.
- Always run an inclusivity check after editing.

## Troubleshooting
- **The draft is too generic** — add more context about the team, mission, or unique perks before regenerating.
- **Generation failed** — wait a moment and try again. If it keeps failing, simplify the prompt.
