# Add screener questions

> Ask the right questions up front so unqualified applicants are filtered out automatically.

## Before you start
- Decide what truly disqualifies an applicant (work authorization, years of experience, location).
- Keep the list short — 3 to 6 questions is usually enough.

## Steps

1. While creating or editing a job, open the **Screener questions** step.
   ![Screener questions step](./assets/01-screener-step.png)
2. Click **Add question** and choose a question type — short text, multiple choice, yes/no, or numeric.
   ![Question type picker](./assets/02-question-types.png)
3. Type the question, set the expected answer if you want it auto-scored, and toggle **Knockout** for must-pass questions.
   ![Configure a question](./assets/03-configure.png)
4. Reorder questions by dragging them, then continue to the next step.

## Tips
- Use **Knockout** sparingly — too many disqualifiers will shrink your applicant pool fast.
- Numeric questions ("Years of experience with X?") are great for ranking, not knocking out.

## Troubleshooting
- **My knockout question isn't filtering anyone** — confirm the expected answer is set correctly. Without one, knockouts won't trigger.
- **Candidates are skipping a question** — only required questions block submission. Toggle **Required** on.
