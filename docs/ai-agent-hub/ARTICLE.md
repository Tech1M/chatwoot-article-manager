# Use the Agent Hub

> The Agent Hub is a library of focused AI agents you can run on demand. Pick an agent, give it the inputs it needs, and review the output.

## Before you start
- Each agent shows its credit cost up front. Make sure your wallet has enough credits.
- Some agents need a specific input — a candidate, a job brief, or pasted text. The agent tells you what it expects.

## Steps

1. Open **AI Tools → Agent Hub** from the sidebar. Agents are grouped by category — sourcing, screening, writing, and more.
   ![Agent Hub catalogue](./assets/01-hub.png)
2. Click an agent to open its run sheet. Fill in the inputs (for example, pick a talent or paste a job description) and review the credit cost.
   ![Agent run sheet](./assets/02-run-sheet.png)
3. Click **Run agent**. The output appears in the panel when it's done. Use the action buttons to copy, save, or apply it to the related record.
   ![Agent output panel](./assets/03-output.png)

## Tips
- Conversational agents let you keep refining the result; single-run agents return one polished answer per run.
- Star the agents you use often — they pin to the top of the hub.

## Troubleshooting
- **Run failed** — check that all required inputs are filled. If it still fails, retry; partial runs don't charge credits.
- **Output looks generic** — give the agent more context in its inputs (a longer brief, more candidate detail).
- **Agent is missing** — your role may not have access to that category. Ask an admin.
