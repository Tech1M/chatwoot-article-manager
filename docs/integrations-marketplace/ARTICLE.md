# Browse the integration marketplace

> The marketplace is where you discover everything Tech1m can connect to, grouped so you can find what you need fast.

## Before you start
- You'll need the **Manage integrations** permission to see the page.

## Steps

1. Open **Settings → Integrations**. Tools you've already connected sit at the top under **Connected**, so they're easy to manage.
   ![Connected section above the marketplace](./assets/01-connected-top.png)
2. Use the category chips — **ATS**, **Calendar**, **Communication**, **Storage**, and more — to narrow the list. Each provider tile shows the logo, name, a short description, and what it does.
   ![Category chips filtering the marketplace](./assets/02-category-filter.png)
3. Search by name if you already know the tool you want. When you find it, click **Connect** to start the setup flow.
   ![Search box with provider results](./assets/03-search.png)

## Tips
- A tool can belong to more than one category — for example, an ATS that also offers messaging will appear under both.
- Tiles you've already connected are hidden from the main grid to keep things tidy. Find them under **Connected** at the top.

## Troubleshooting
- **The grid is empty** — check the category filter; you may have one selected with no providers. Clear it to see everything.
- **A tool you expected isn't here** — it may not be supported yet. Submit a request via [Request a new integration](../integrations-request-new/ARTICLE.md).
- **"We couldn't load the marketplace"** — refresh the page. If it persists, contact support.
