# Generate expense reports

> Build a report of expenses across people, projects, or time periods and export it for accounting or a project review.

## Before you start
- The expenses you want to include are submitted (and ideally approved).
- You know the date range, project, or person you want to report on.
- You have an export-friendly tool such as a PDF viewer or spreadsheet to open the file.

## Steps

1. Open **Operations → Expenses → Reports** and click **New report**.
   ![Expense reports page](./assets/01-reports-page.png)
2. Set the filters: date range, project, expense type, person, and status. The preview updates with the totals and a category breakdown.
   ![Report filters with category breakdown preview](./assets/02-filters-preview.png)
3. Choose **Export PDF** for a polished summary, or **Export CSV** to keep working with the data in a spreadsheet.
   ![Export options on the report preview](./assets/03-export-options.png)
4. Save the report — exported files include the totals, the category chart, and the line-by-line list of expenses.

## Tips
- Save common report filters as a named view so you can rerun them at month-end.
- Use the CSV export when you need to feed data into your accounting system.

## Troubleshooting
- **Totals look low** — check whether you filtered out non-approved expenses.
- **Export fails** — narrow the date range; very large reports can time out.
