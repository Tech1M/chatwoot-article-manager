# Track gift-card orders

> Use the order history to monitor delivery, troubleshoot failed orders, and reconcile spend.

## Before you start
- Place at least one gift card order so there's something to track.

## Steps

1. Open **Perks → Orders**. The list shows every gift card your company has ordered, newest first.
   ![Orders list](./assets/01-orders-list.png)
2. Use the filters at the top to narrow by **Status** (Pending, Delivered, Failed) or by **Merchant**. Sort by date or value with the column headers.
3. Click into any order to see the full detail — recipient, amount, payment method, voucher status, and the delivery history.
   ![Order detail](./assets/02-order-detail.png)
4. From the detail page you can **Resend** the delivery email if the recipient lost the original, or **Retry** an order that failed at the merchant.

## Tips
- Export the filtered list to share with finance for reconciliation.
- Pending orders normally clear within a few minutes. If one is stuck for more than an hour, open it to check the merchant status.

## Troubleshooting
- **Order stuck on Pending** — the merchant hasn't confirmed yet. Wait a few minutes; if it still doesn't move, contact support with the order ID.
- **Voucher code doesn't work for the recipient** — open the order, copy the code from the latest delivery, and resend. Some merchants reissue codes after a failed first attempt.
