# Add candidates via the Chrome extension

> Save candidate profiles to Tech1m while you browse, without copy-pasting.

## Before you start
- Install the Tech1m Chrome extension from the Chrome Web Store.
- Sign in to Tech1m in the extension using your business account.
- Create at least one talent pool.

## Steps

1. Visit a candidate's public profile (for example, on LinkedIn). Click the Tech1m icon in your Chrome toolbar.
   ![Extension icon](./assets/01-extension-icon.png)
2. The extension reads the visible profile and shows a preview of the captured details.
   ![Profile preview](./assets/02-profile-preview.png)
3. Pick the talent pool to save the candidate into. You can also add tags or a short note before saving.
   ![Choose pool](./assets/03-choose-pool.png)
4. Click **Save to pool**. You will see a confirmation, and the candidate is now searchable in your CRM.
   ![Saved confirmation](./assets/04-saved-confirmation.png)

## Tips
- If the candidate already exists in your CRM, the extension will offer to add them to the pool rather than create a duplicate.
- Use the extension on a candidate's "About" page rather than search results so it can capture their full background.

## Troubleshooting
- **The extension says "not signed in"** — click the icon and sign in with the same account you use on Tech1m.
- **No data is captured** — refresh the page, then open the extension again. Some sites lazy-load profile content.
- **Wrong company workspace** — switch workspaces from the extension's settings menu.
