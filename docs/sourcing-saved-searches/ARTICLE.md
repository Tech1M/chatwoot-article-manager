# Save and reuse searches

> Keep your best searches one click away instead of rebuilding them from scratch.

## Before you start
- Run a search first — you can only save a search after it returns results.
- You need permission to create AI Sourcing queries.

## Steps

1. After running a search, click **Save search**. Give it a name that describes the role or campaign.
   ![Save search button](./assets/01-save-button.png)
2. Open **AI Sourcing → Saved searches** to see everything you've saved. Recent searches you ran but didn't save are listed separately.
   ![Saved searches list](./assets/02-saved-list.png)
3. Click any saved search to rerun it instantly with the latest results. Adjust filters from there if you want a variation.
4. Remove a saved search you no longer use with the **Delete** action — this won't affect any candidates already added to talent pools or sequences.

## Tips
- Name searches by hiring need, not by date — *"Senior FE - EU remote"* is easier to scan than *"April search 3"*.
- Rerun a saved search before each weekly sourcing review to surface newly indexed candidates.

## Troubleshooting
- **Save button is disabled** — your search has no text and no filters. Add at least one before saving.
- **A saved search returns nothing now** — the criteria may be too tight. Open it, loosen one filter, and resave.
