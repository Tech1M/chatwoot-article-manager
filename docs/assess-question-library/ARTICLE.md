# Use the question library

> The question library is your team's shared bank of vetted assessment items. Pull from it, save your best ones to it, and stop reinventing each round.

## Before you start
- An assessment open in the builder, or permission to manage the library.

## Steps

1. In the assessment builder, click **Add from library**. Filter by skill, role, difficulty, or question type.
   ![Question library filters](./assets/01-library-filters.png)
2. Preview a question to see the prompt, options, correct answer, and where it has been used before. Select one or many and click **Add to assessment**.
   ![Question preview](./assets/02-question-preview.png)
3. To save a question for reuse, open the question's menu and choose **Save to library**. Tag it with the relevant skill and difficulty so others can find it.
   ![Save to library option](./assets/03-save-to-library.png)
4. Manage all saved questions from **Assessments → Library**, where you can edit, archive, or merge duplicates.

## Tips
- Tag aggressively — most teams find questions by skill and difficulty, not by title.
- Mark questions you suspect have leaked as **Retired** instead of deleting them, so old assessments still render.

## Troubleshooting
- **Question not appearing in search** — make sure it has a skill tag. Untagged questions are hidden from the picker by default.
- **Can't edit a question** — only the author or workspace admins can edit. Duplicate it to make your own copy.
