# Run an inclusivity analysis

> Spot biased phrasing in your job description before it goes live.

## Before you start
- Have a near-final job description ready — analysis works best on complete copy.
- Be open to rewriting flagged phrases; the tool suggests alternatives, not rules.

## Steps

1. While editing a job, open the description step and click **Run inclusivity analysis**.
   ![Run analysis button](./assets/01-run-button.png)
2. The panel highlights flagged phrases and shows the bias category — gendered, age-related, or exclusionary language.
   ![Flagged phrases](./assets/02-flagged.png)
3. Click any suggestion to apply it directly, or rewrite the sentence yourself.
   ![Apply suggestion](./assets/03-suggestion.png)
4. Re-run the analysis until the description scores well, then continue.

## Tips
- Watch out for "rockstar", "ninja", and "young, energetic" — common phrases that narrow your applicant pool.
- A short, clear description usually scores better than a long, jargon-heavy one.

## Troubleshooting
- **Analysis didn't return any suggestions** — your description is short. Add more content and try again.
- **A suggestion doesn't fit your context** — skip it. The tool is advisory.
