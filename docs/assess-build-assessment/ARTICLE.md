# Build an assessment

> Mix questions, code challenges, and take-home tasks into one branded assessment you can reuse across jobs.

## Before you start
- A clear idea of the role and the skills you want to measure.
- Permission to create assessments in your workspace.

## Steps

1. From the sidebar, open **Assessments** and click **New assessment**. Give it a name and a short description candidates will see.
   ![New assessment dialog](./assets/01-new-assessment.png)
2. In the builder, add components one at a time — multiple-choice questions, skill tests, code challenges, take-home assignments, or psychometric tests. Drag components to reorder them.
   ![Assessment builder canvas](./assets/02-builder-canvas.png)
3. Set the **time limit**, **passing score**, and whether candidates can pause or retake. You can also turn on AI-assisted evaluation for written answers.
   ![Assessment settings panel](./assets/03-settings.png)
4. Preview the candidate experience, then click **Publish**. Draft assessments stay private until you publish them.

## Tips
- Keep total time under 45 minutes — completion rates drop sharply beyond that.
- Reuse a published assessment across multiple jobs instead of rebuilding it each time.

## Troubleshooting
- **Can't publish** — every component needs at least one question and a point value. The builder will highlight what's missing.
- **Component greyed out** — your plan may not include that component type. Check the badge on the component card.
