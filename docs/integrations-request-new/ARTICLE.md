# Request a new integration

> Don't see the tool you use? Send us a request — we prioritize new integrations based on what customers ask for.

## Before you start
- Double-check the marketplace under different categories. The tool may already exist under a slightly different name.
- Have the provider's website or product page handy. It helps us identify the right one.

## Steps

1. Open **Settings → Integrations** and click **Request integration** at the top of the page.
   ![Request integration button](./assets/01-request-button.png)
2. Fill in the request form: the tool's name, a link to its website, and the use case you need (importing candidates, syncing calendars, posting updates, and so on).
   ![Request integration form](./assets/02-request-form.png)
3. Submit the request. Our team gets a notification and will follow up by email if we need more details.
   ![Request submitted confirmation](./assets/03-submitted.png)

## Tips
- The clearer the use case, the better. "We want to push hired candidates into our HRIS" is more useful than "Add HRIS support."
- If multiple teammates need the same integration, ask each of them to submit a request — it helps us gauge demand.

## Troubleshooting
- **The form won't submit** — make sure required fields aren't blank, then try again.
- **No reply yet** — we don't reply to every request, but high-demand integrations get prioritized in upcoming releases. Check the marketplace periodically.
