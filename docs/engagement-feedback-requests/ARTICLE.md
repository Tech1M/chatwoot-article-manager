# Send a feedback request

> Ask a specific person or group for feedback on a topic, role, or teammate without setting up a full survey.

## Before you start
- Know who you want to ask and what you’re asking about.
- Make sure the recipient exists in your directory or has a valid email on file.

## Steps

1. Open **Engagement Centre** and click **Feedback requests** in the side menu.
   ![Feedback requests inbox](./assets/01-feedback-inbox.png)
2. Click **New request**, then choose a request type — peer feedback, manager check-in, candidate experience, or custom.
3. Pick the recipients. You can add multiple people; each gets their own private link.
   ![Recipient picker](./assets/02-recipients.png)
4. Add the prompts or questions you want them to answer. Keep it focused — 3 to 5 prompts works best.
5. Choose whether responses should be anonymous and set a due date.
6. Click **Send**. You’ll see each recipient’s status (pending, viewed, completed) on the request page.
   ![Request status tracking](./assets/03-request-status.png)

## Tips
- Add a short personal note explaining why their input matters — it lifts response rates.
- Use a recurring request for regular check-ins instead of sending a new one each time.

## Troubleshooting
- **Recipient didn’t get the email** — check their address in the directory and resend from the request page.
- **Request stuck on Pending** — the link expires after the due date; extend or resend it.
