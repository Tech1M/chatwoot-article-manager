# Add notes and view activity

> Build a shared memory of every candidate so the next teammate to reach out has the full picture.

## Before you start
- Open the candidate's profile from any pool or search result.

## Steps

1. On the candidate profile, switch to the **Notes** tab.
   ![Notes tab](./assets/01-notes-tab.png)
2. Click **Add note**, type your note, and save. Use notes for context like "Met at AfricaTech 2025, looking for senior roles in Q3".
   ![Write a note](./assets/02-write-note.png)
3. To edit or delete a note, use the menu on each note. Only the author and admins can change a note.
4. Switch to the **Activity** tab to see a timeline of every action: when the candidate was added, who tagged them, every SMS, email, and pool change.
   ![Activity timeline](./assets/03-activity-timeline.png)

## Tips
- Notes are shared across your company. Anyone with CRM access can read them, so keep them professional.
- Mention a teammate in a note with `@name` to send them a notification.
- Use the activity tab before reaching out so you do not repeat someone else's outreach.

## Troubleshooting
- **I cannot edit a note** — only the author or an admin can edit a note. Ask the original author or an admin to update it.
- **Activity is empty** — the candidate may have just been added. Activity entries appear as soon as your team interacts with the profile.
