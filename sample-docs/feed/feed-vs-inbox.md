## When to use Feed

Use the **Feed** when you want to share something with the whole team:

- Company announcements
- Project updates
- Team celebrations
- Policy changes

## When to use Inbox

Use the **Inbox** for direct, private conversations:

- One-on-one discussions
- Sensitive topics
- Quick questions to a specific person
- Document sharing with individuals
